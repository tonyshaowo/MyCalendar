# MyCalendar 專案：Google 日曆忙碌狀態顯示頁面

##  專案簡介

這個儲存庫包含一個簡單的 HTML 頁面，用於**內嵌（Embed）**我的 Google 日曆，並將其顯示權限設定為 **「僅顯示有空/忙碌（隱藏詳細資料）」**。

本頁面採用**響應式設計（RWD）**，確保在各種裝置（手機、平板、電腦）上都能以滿版模式正確顯示。

##  網站連結

您可以透過以下連結查看已發布的日曆忙碌狀態頁面：

👉 **[點此查看我的日曆](https://tonyshaowo.github.io/MyCalendar/)** 👈

##  技術說明

* **目的：** 安全地分享我的行程狀態，同時保護活動細節的隱私。
* **技術：** HTML5, CSS3。
* **部署平台：** GitHub Pages。

### 隱私權設定重點

* 內嵌程式碼中的 Google 日曆權限已在後台設定為 **「僅顯示有空/忙碌」**。
* 訪客**無法**透過調整網頁參數來查看我的活動標題、描述、地點等詳細資訊。
* 頁面只會顯示特定時段我是「有空」還是「忙碌」。

## 📌 如何使用 (開發者參考)

如果您想使用這個模板：

1.  **替換日曆 ID (重要！)**：
    * 進入您的 Google 日曆 **「設定與分享」** > **「整合日曆」** 區塊。
    * **強烈建議：** 不要直接使用「日曆 ID」欄位中顯示的 Gmail 帳號。
    * 點擊內嵌程式碼下方的 **「自訂」** 按鈕 。
    * 在自訂頁面中，調整顯示參數後，將右側產生的 **`<iframe>` 程式碼**中的 `src` 連結複製出來。這個連結會使用一串**不包含您 Gmail 帳號的「群組日曆 ID」**的亂碼，從而保護您的個人資訊。
2.  **更新程式碼：** 將步驟 1 中複製的 `src` 參數替換到 `index.html` 檔案的 `<iframe src="...">` 連結中。
3.  **確認權限：** 確保您的 Google 日曆後台的「存取權限」已設定為 **「僅顯示有空/忙碌（隱藏詳細資料）」**。
